## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| Adventure Green | `#2E7D32` | Primary, buttons, active states |
| Green Light | `#60AD5E` | Hover, light backgrounds |
| Amber | `#FFA000` | Accent, ratings, badges |
| Amber Dark | `#C67100` | Warning text |
| Surface Light | `#FBFBFB` | Background (light mode) |
| Surface Dark | `#1E1E1E` | Background (dark mode) |
| Error | `#D32F2F` | Danger, delete |

## Typography

- **Headings:** Inter / SF Pro (iOS)
- **Body:** Inter / SF Pro Text
- **Monospace:** JetBrains Mono (for stats/code)
