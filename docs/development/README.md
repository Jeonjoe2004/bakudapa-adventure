# Development

Panduan development untuk kontributor Bakudapa Adventure.

## Setup

1. Clone repo
2. Install dependencies per platform (lihat README root)
3. Firebase: setup project, download config files
4. Seed data: `node scripts/seed-firestore.mjs`

## Conventions

- Android: MVVM + MVI, Kotlin, Jetpack Compose
- iOS: SwiftUI + MVVM
- Admin Web: React + TypeScript + Tailwind
- Git: conventional commits
